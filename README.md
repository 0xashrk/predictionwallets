# Wallet Tracker

A wallet tracking application built with React, TypeScript, and Supabase.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Supabase

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun

### Installation

```sh
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Fill in your Supabase credentials in .env
```

### Development

```sh
# Start the dev server (runs on port 8080)
npm run dev
```

### Build

```sh
npm run build
```

### Test

```sh
npm test
```
