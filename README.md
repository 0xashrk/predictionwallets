# Wallet Tracker

A wallet tracking application built with React, TypeScript, and Supabase.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Supabase

## Getting Started

### Prerequisites

- Node.js (v18+)
- bun

### Installation

```sh
# Install dependencies
bun install

# Copy environment variables
cp .env.example .env

# Fill in your Supabase credentials in .env
```

### Development

```sh
# Start the dev server (runs on port 3001)
bun run dev
```

### Build

```sh
bun run build
```

### Test

```sh
bun test
```
